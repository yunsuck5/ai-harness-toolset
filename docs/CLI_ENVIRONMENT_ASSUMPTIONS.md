# CLI Environment Assumptions

The following may exist in the user's external environment but are not core dependencies of this toolset:

- Claude Code CLI
- Codex CLI
- Superpowers
- Serena MCP
- Sequential Thinking MCP
- Context7 MCP
- Git
- Node.js
- PowerShell

This toolset does not install, configure, or mutate global settings for any of the above. It does not require any of them to function. Where present, they may be invoked manually by the user via documented snippets.
